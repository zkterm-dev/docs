## Security Updates

### CVE-2024-001 (Fixed in v1.5.1)

**Severity**: Low  
**Impact**: IPFS gateway timeout could cause UI freeze  
**Fix**: Added timeout handling and retry logic

### CVE-2024-002 (Fixed in v1.4.3)

**Severity**: Medium  
**Impact**: Proof verification could accept invalid public signals  
**Fix**: Enhanced validation in proof verification
